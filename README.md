# reserva de salas alura desafio primeiro checkPoint

este projeto é sobre gerenciamento de salas com algumas validações para impedir conflitos.

## Tecnologias

- Linguagem: java 17
- Framework: spring boot 4.0.2
- Persistência: Spring Data JPA/Hibernate
- Build: Maven
- Banco de dados: MySQL

## Como rodar a aplicação

- O projeto utiliza **MySQL** que precisa ser configurado em application.properties
- Clone o repositório: `git clone https://github.com`
- Certifique-se de ter o **Java 17+** e **Maven** instalados.
- Execute o comando: `mvn spring-boot:run`
- A API estará disponível em: `http://localhost:8080/api/v1`

## Testes Automatizados
O projeto conta com uma suíte de testes unitários e de integração utilizando **JUnit 5**, **Mockito** e **MockMvc**.
